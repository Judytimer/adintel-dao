> Transform advertising data into collective intelligence. Chrome users earn rewards for contributing ad insights while businesses get affordable competitive analysis.

## 🎯 Project Overview

AdIntel DAO is a decentralized network that revolutionizes market intelligence by:
- **Collecting**: Public advertising data through a privacy-compliant Chrome extension
- **Rewarding**: Users with tokens for contributing valuable market data
- **Providing**: Affordable competitive intelligence to small and medium businesses

### Key Value Propositions
- **For Users**: Earn $5-20/month passively while browsing + free competitive insights
- **For Businesses**: 80% cheaper than traditional tools ($30 vs $150/month)
- **For the Ecosystem**: Democratize market intelligence and break data monopolies

## 🌟 Features

### Chrome Extension
- 🔍 **Automatic Ad Detection**: Detects ads across major platforms (Facebook, Google, etc.)
- 🔒 **Privacy-First**: Zero personal data collection, only public ad content
- 💰 **Passive Earning**: Users earn tokens for unique ad contributions
- 📊 **Real-time Analytics**: Instant insights into competitor strategies

### Business Dashboard
- 🎯 **Competitor Analysis**: Track competitor ad campaigns and strategies
- 📈 **Market Trends**: Identify trending ad formats and messaging
- 💡 **Creative Inspiration**: Browse successful ad creatives by industry
- 📋 **Export Tools**: Download insights for campaign planning

### Blockchain Integration
- 🌐 **Vana Network**: Built on Vana's data DAO infrastructure
- 🎖️ **Token Rewards**: Contributors earn tokens based on data quality
- 🏛️ **Governance**: Community-driven decision making
- 🔐 **Data Ownership**: Users retain rights to their contributed data

## 🚀 Quick Start

### For Users (Extension)
1. **Install**: Download from Chrome Web Store (coming soon)
2. **Browse**: Continue normal browsing - extension works automatically
3. **Earn**: Accumulate tokens for each unique ad detected
4. **Redeem**: Use tokens for premium features or convert to cash

### For Businesses (Dashboard)
1. **Sign Up**: Create account at [adintel-dao.com](https://adintel-dao.com)
2. **Search**: Enter competitor names or keywords
3. **Analyze**: View comprehensive ad intelligence reports
4. **Export**: Download insights for your marketing team

### For Developers
```bash
# Clone the repository
git clone https://github.com/adintel-dao/adintel-dao.git
cd adintel-dao

# Install dependencies
npm install

# Run tests
npm test

# Start development server
npm run dev

# Build Chrome extension
npm run build:extension
```

## 🛠️ Technology Stack

### Frontend
- **Chrome Extension**: Vanilla JavaScript, HTML5, CSS3
